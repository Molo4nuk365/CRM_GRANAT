# CRM Jewelry Workshop

CRM-система для ювелирной мастерской с управлением заказами, пользователями, материалами и товарами.

## Технологии
- Версия .NET 9 
- Visual Studio 2026 года - бэкенд 
- Visual Code (версия 1.119.0) 2026 года - фротэнд 

### 1. Установочные пакеты и шаблон проекта Visual Studio
- (Веб-приложение ASP.NET Core (модель-представление-контроллер))
- BCrypt.Net-Next  (версия 4.2.0)
- Microsoft.AspNetCore.Authentication.JwtBearer (версия 9.0.0)
- Microsoft.EntityFrameworkCore.Design  (версия  9.0.14)
- Microsoft.EntityFrameworkCore.Sqlite  (версия  9.0.14)

### 2. Защита данных 
- JWT (аутентификация)
- BCrypt (хеширование паролей)

### 3.База данных 
- SQLite (хранит всю информацию на одном устройстве в одном файле подходит для небольших проектов)

### 4. Фронтенд
- Bootstrap 5.3.0 – сетка, кнопки, карточки, навигация, готовые модальные окна (без jQuery)
- Font Awesome 6.4.0 – иконки (корзина, луна/солнце, шестерёнки и т.д.)
- Google Fonts – шрифт Montserrat
- Intersection Observer API – плавная анимация карточек товаров при скролле
- Кастомные CSS-переменные – поддержка светлой и тёмной темы с переключением

### 5.Docker Desctop
- Docker Desktop (контейнеризация)

## Роли и права доступа
- **admin** – Полный доступ: управление пользователями, ролями, материалами, товарами, назначения ювелира
- **manager** – Просмотр всех заказов, назначение ювелира, изменение статуса
- **jeweler** – Просмотр и выполнение своих заказов, установка срока, изменение статуса
- **client** – Создание заказов, оплата, просмотр своих заказов
